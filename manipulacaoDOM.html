Manipulação da DOM

Document Object Model (DOM) é um objeto que contém a representação da página web sendo exibida. A DOM parte da especificação HTML, mas pode ser alterada por alguma linguagem de programação ao longo da interação com a página.

Nesse exemplo, criamos uma lista de tarefas, que contém uma função JavaScript que é acionada no cique do botão para adicionar uma tag (representando uma nova tarefa) à DOM
