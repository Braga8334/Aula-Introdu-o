Seletores CSS

Você pode atribuir estilos CSS às tags HTML de duas formas.

  1. Você pode definir o estilo inline com a propriedade style:
    Cor de fundo aquamarine definida inline

  2. Você pode (e é uma boa prática) separar os estilos num arquivo style.css e vinculá-lo ao html com a link (veja acima de title)

O CSS aplica aos propriedades de acordo com a seguinte ordem de prioridade:

  • inline - propriedades definidas inline sobrescrevem qualquer outra
  • id - propriedades definidas por id da tag
  • class - propriedades definidas pela classe da tag
  • tag - propriedades definidas para todos elementos de uma tag
  • global - propriedades definidas para todos elementos de todas as tags
